# Ayat
Ayat

خطة شاملة لتطبيق إسلامي كامل Offline مع تحميل البيانات عند الاتصال بالإنترنت
1️⃣ الرؤية العامة

هدف التطبيق:

منصة إسلامية شاملة لمواقيت الصلاة، الأذكار، القرآن الكريم، التفاسير، التسبيح، القبلة، والتقويم الهجري.

يعمل Offline بالكامل بعد تحميل المحتوى عند الاتصال بالإنترنت.

جميع البيانات مخزنة محليًا في قاعدة بيانات Room.

الخصوصية: لا تتبع، لا إعلانات، كل شيء على الجهاز.

2️⃣ نطاق المميزات
أ) مواقيت الصلاة

حساب تلقائي حسب الموقع الجغرافي (GPS) أو يدويًا.

طرق حساب متعددة: Umm Al-Qura، MWL، Egyptian، etc.

أذان كامل بصوت مؤذن قابل للاختيار.

عداد تنازلي للصلاة القادمة.

تنبيهات قبل الصلاة (يمكن ضبطها).

حفظ التفضيلات محليًا في Room.

ب) القرآن الكريم

نص كامل بالرسم العثماني.

تقسيم حسب: السور، الأجزاء، الأحزاب.

البحث النصي داخل القرآن.

تظليل الآية عند التشغيل الصوتي.

تحميل صوتيات السور عند الاتصال بالإنترنت وحفظها محليًا.

تشغيل الصوت Offline لاحقًا.

اختيار القارئ من قائمة متعددة (عبد الباسط، مشاري العفاسي، ماهر المعيقلي، …).

ج) التفاسير

دعم تفاسير متعددة (ابن كثير، الطبري، السعدي…).

عرض توازي للآية والتفسير.

تحميل التفسير عند الحاجة وحفظه Offline.

إمكانية البحث في التفاسير محليًا.

د) الأذكار والسبحة

أذكار الصباح والمساء، بعد الصلاة، النوم، رقية شرعية.

عداد تسبيح تفاعلي.

متابعة ورد شخصي يومي.

حفظ التقدم في Room.

إشعارات التذكير بالأذكار.

هـ) القبلة

حساب الاتجاه محليًا باستخدام Sensor API + GPS.

عرض بوصلة تفاعلية.

Offline بعد أول تحميل البيانات.

و) التقويم الهجري

عرض الأيام والمناسبات (رمضان، عيد الفطر، عيد الأضحى…).

حساب الصيام المستمر.

حفظ التواريخ والملاحظات محليًا.

ز) ميزات متقدمة

وضع رمضان: عداد الإفطار، ختمة رمضان، أذكار خاصة.

وضع قيام الليل: تنبيه تلقائي، عداد جلسة.

برنامج ورد شخصي ذكي: متابعة ختمات، سجل التقدم.

وضع “الخلوة” أو “الاعتكاف”: إخفاء كل الإشعارات وتشغيل ذكر متكرر.

مؤقت الصلاة القصيرة (السفر).

دعم القراءات المختلفة: حفص، ورش، قالون.

ح) تجربة القراءة والقراءة الصوتية

تظليل النص أثناء الصوت.

وضع مقارنة صوتية بين القراء.

تغيير سرعة التشغيل.

Repeat آية / سورة / جزء.

ط) Gamification ومتابعة التقدم

متابعة ختمة القرآن.

متابعة الأذكار اليومية.

مؤشرات التزام المستخدم.

إحصائيات شخصية محفوظة محليًا.

3️⃣ المعمارية التقنية
Frontend

Flutter (Android & iOS)

State Management: Riverpod أو Bloc

Modular Feature-based Architecture

UI/UX: تصميم هادئ وفخم مع خط عربي مميز

Widgets مخصصة لكل ميزة

Database محلي

Room Database:

الجداول: surahs، ayahs، tafsirs، reciters، audio_files، azkar، user_progress، settings

جميع المحتويات والملفات الصوتية مخزنة داخليًا

دعم Lazy Loading وQuery Optimization

Storage: الملفات الصوتية محفوظة محليًا على الجهاز (Internal Storage أو App Directory)

Audio Playback

Just Audio أو أي مكتبة تدعم تشغيل Offline.

دعم تكرار، تشغيل متواصل، تظليل النص أثناء التشغيل.

Background Services

WorkManager أو AlarmManager:

تنبيهات الصلاة

تنبيهات الأذكار

تشغيل أذان أو صوت تلقائي

Networking

تحميل البيانات عند الاتصال بالإنترنت:

القرآن (نصوص + صوتيات)

التفاسير

الأذكار (إذا تم تحديث جديد)

التخزين المحلي في Room بعد التحميل.
